                            Java 基本语法 #

                            访问控制 #

                            • private: 同一个类
                            • 无 (default): 本包下的类
                            • protected: 本包下类，或者不同包下的子类
                            • public: 所有类
                            ______________________________________________________________

|           │       │         │(same pkg)│(diff pkg)│        |
|───────────┼───────┼─────────┼──────────┼──────────┼────────|
|public     │   +   │    +    │    +     │     +    │   +    |
|───────────┼───────┼─────────┼──────────┼──────────┼────────|
|protected  │   +   │    +    │    +     │     +    │        |

|private    │   +   │         │          │          │        |
|___________|_______|_________|__________|__________|________|
 + : accessible         blank : not accessible
